# 郵便番号検索アプリ

## 概要
郵便番号を入力すると、対応する住所を自動で表示するシンプルな検索アプリです。

## 使用技術
- HTML / CSS / JavaScript
- Vue.js（CDN）
- jQuery
- CSVデータを使用した検索処理

## 工夫した点
- CDN版Vueを使用し、ビルド不要で動作する構成にした
- CSVデータを非同期で読み込み、検索処理を実装
- ポートフォリオ掲載を想定し、シンプルなUIにまとめた

## 動作確認
- ローカルサーバー（Live Server 等）
- サーバーにアップロードしての動作確認

## 補足
※ ローカルで直接HTMLを開くと、ブラウザの仕様によりCSV読み込みでエラーが発生します。
